# Search Heist AI

AI-Powered Marketing Intelligence & Search Merchandising Agent

---

# Overview

Search Heist AI works like “The Professor” from Money Heist for marketing analytics and search intelligence.

Upload campaign or search datasets and interact with an AI agent that can:

* Analyze campaign performance
* Detect weak campaigns
* Find wasted spend
* Identify ROAS decline
* Detect KPI anomalies
* Generate executive summaries
* Answer marketing questions
* Suggest optimization recommendations
* Support search merchandising analysis

# Installation

## Clone Repository

```bash
git clone https://github.com/inteshabul-haque/Search_Heist_AI.git
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run main.py
```
